# 🚀 DevOps CI/CD Project – Jenkins + SonarQube (Full Stack Application)

## 📌 Project Overview

This project demonstrates a complete **end-to-end DevOps workflow** for a full-stack application with **code quality analysis using SonarQube**.

It includes:

- Frontend (React)
- Backend (Spring Boot)
- Database (AWS RDS)
- CI/CD Pipeline using Jenkins
- Code quality and security analysis using SonarQube

The pipeline automates code checkout, backend build, SonarQube analysis, and quality gate verification.

---

## 🏗️ Tech Stack

- **Frontend**: React (Vite)
- **Backend**: Spring Boot (Java 17)
- **Database**: MariaDB (AWS RDS)
- **CI/CD Tool**: Jenkins
- **Code Quality Tool**: SonarQube
- **Cloud Services**: AWS EC2, AWS RDS
- **Build Tool**: Maven
